## The C5B Calling Card

C5B / Golden Mark is the centerpiece because it shows the research method:

1. take a hard, ambiguous domain
2. define what evidence would count as progress
3. build a baseline route
4. run the architecture-on route
5. compare behavior, stability, source accuracy, and output discipline
6. preserve receipts
7. label claims, evidence, gaps, and next gates
8. iterate until the route improves

This is the transferable capability. The point is not that Tris knows one
topic. The point is that Renaissance Field Lite can build the research machine
around a frontier problem.
